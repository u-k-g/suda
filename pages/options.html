Keybinding mode


Helix mode uses g for goto commands, Ctrl-w for tabs, Space for pickers, and selection-first text navigation. Custom mappings below are applied after the selected mode's defaults.

Custom key mappings

Example syntax:
map j scrollDown
map z2 setZoom level=2
unmap j
unmapAll

Custom search engines

Add search-engine shortcuts to the Vomnibar. Format:
a: http://a.com/?q=%s
b: http://b.com/?q=%s description
" this is a comment
# this is also a comment
%s is replaced with the search terms.

New tab URL

The page to open for Chrome's Ctrl-T/Cmd-T shortcut, the new-tab button, and Vimium's "create new tab" command. The browser's default page is protected, so Vimium commands do not run there.
